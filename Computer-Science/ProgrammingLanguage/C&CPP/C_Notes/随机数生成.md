**rand()**
是伪随机
通过将种子进行一系列计算，获取一个伪随机的数列

























头文件
```c
#include<stdlib.h>
```

使用 rand()  产生随机数
RAND_MAX取决于系统和编译器
使用
```c
printf("%ld\n",RAND_MAX);
```
可查看RAND_MAX大小

rand为伪随机
会打印计算机事先生成的随机数列
传入一个种子
以此种子进行一系列计算
从而获得一串伪随机的数列

srand(seed：) 
传入一个种子来进行伪随机计算


头文件
```c
#include<time.h>
```

从而可以调用time函数
其参数为一个指针
不想做修改直接填 NULL

借助time函数
从而保证rand使用的种子不同
从而达到随机效果

通过取模运算
可以达到限制随机数范围的效果
范围是多少就 % 多少 + 多少
