***表格***
使用**竖线符|** 来分隔每一列
使用**竖线符|和减号-在第二行进行|-------|** 式书写来分隔表头，**减号-** 至少有三个，通过**冒号：** 来达到不同的效果
1. **:---** 达到左对齐效果
2. **:---:** 来达到剧中对齐效果
3. **---:** 来达到右对齐效果
从第三行开始为表格内容
**示例**

| 值      | 含义                  | 在条件表达式中 |
|---------|-----------------------|----------------|
| `true`  | 真值                  | true          |
| `false` | 假值                  | false         |
| `nil`   | 表示“无”或“不存在”    | false         |
| 其他    | 任何非 false/non-nil 对象 | true          |
