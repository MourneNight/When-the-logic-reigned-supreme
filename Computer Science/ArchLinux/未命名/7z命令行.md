1. 压缩文件/目录
```zsh
7z a archieve.7z file1 flie2
7z a archieve.7z dir/

#压缩为zip
7z a -tzip archive.zip file
#添加密码
7z a -p123456 archive.7z file
```
2. 解压文件
```zsh
7z x archive.7z #当前目录
7z x archive.7z -o/path/ #指定目录
7z x -p123455 archive.7z  #带密码

# 命令 x 意味着保留文件目录结构
# x 替换 e 可无视文件目录结构
```
3. 列出压缩包内文件
```zsh
7z l archive.7z
```
4. 删除压缩包内文件
```zsh
7z d archive.7z file1
```
5. 测试压缩包完整性
```zsh
7z t archive.7z
```
6. 分卷压缩
```zsh 
7z a -v100M archive.7z
large_file.dat
#按100MB分卷
```
