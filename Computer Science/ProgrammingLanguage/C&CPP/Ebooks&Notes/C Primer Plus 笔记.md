
# 初识C语言

***起源***
1972，在汤普森发明的B语言基础上，贝尔实验室的丹尼斯·里奇（Dennis Ritch）和肯·汤普逊（Ken
Thompson）在开发UNIX操作系统时设计了C语言。

### Why use C?
1. 用C的程序更易懂、可靠（？）
2. 高效，具备微调控制能力，从而获取最大运行速度或最有效地使用内存
3. 可移植性好，且，强大而灵活
4. 面向程序员（Ruby）

### C语言标准
最初公认的C标准：K&R
ANSI/ISO C标准，也叫C89;此外，还有C99,C11等
